Options - When Weather Changes

This option allows you to decide what happens to the icon when the weather changes. Any one of the following events causes the weather to be seen as having changed:
  1. The postcode
  2. The weather description
  3. The temperature
  4. The "feels like" temperature
The choices for what to do are as follows.



Option Description
Flash Continuously The icon will flash continuously every second (or whatever frequency is specified in the options) from light to dark. The flashing will NOT cease when the mouse is moved over the icon.
Flash The icon will flash every second (or whatever frequency is specified in the options) from light to dark. The flashing will cease when the mouse is moved over the icon.
Change Once The icon will change from "dark" to "light". The icon will change back to "dark" when mouse is moved over the icon.
No Change The icon colour is not changed.


The terms "light" and "dark" refer to the colours of the icons. For a complete description of what each of the colours are at the various temperatures, see Colour Scheme. For more information on what happens to the icon in the various scenarios that UK Weather encounters, see UK Weather Desktop.
