Options

The Options form is where all the configuration of UK Weather is done. As of version 0.61, there are eight options to configure. They appear on three subforms in Options:







    On the first subform, which can be accessed by clicking on "User Interface", the following option can be configured:
  1. Temperature Scale


    2. On the second subform, which can be accessed by clicking on "Updating", the following options can be configured:
  2. Postcode
  3. Update Interval
  4. When the weather changes
  5. Icon Flash Frequency


    6. On the third subform, which can be accessed by clicking on "Startup", the following options can be configured:
  6. Run at system startup
  7. Notify user if new version is found
  8. Check for new version and URL
Clicking "OK" accepts any changes made and clicking "Cancel" cancels any changes entered via this screen. Clicking "OK and Update" accepts any changes made and then forces UK Weather to try to update the data. UK Weather will behave exactly as if it were due to update the weather data. Also, the timer is reset, when "Update Now" is chosen. For example, if the update interval is every 10 minutes and the "Update Now" option is chosen after 3 minutes, UK Weather will update immediately and then after 10 minutes, not 7 minutes.
