Options - Startup

Tick this tickbox to cause UK Weather to start when the user logs in. It stores the setting in the following location: HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run in a value (not surprisingly) called "UKWeather". Note that I'm using the current user rather than HKEY_LOCAL MACHINE hive.

If the option is unticked, it removes the above registry entry altogether.
