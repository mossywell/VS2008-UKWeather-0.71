System Tray Icon Symbols

In general, the system tray icon shows the current temperature. However, as always, it's a little more complex than that! What the icon shows actually depends on a number of factors.
  • The UK Weather options
  • The temperature
  • Whether the temperature or the description of the weather has changed
  • Whether there is an Internet connection when UK Weather tries to obtain data
  • Whether the postcode entered is valid
  • What was previously displayed as the icon
