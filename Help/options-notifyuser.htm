Options - Notify User...

By default, UK Weather checks 3 things when it starts up:
  1. Whether there is a new version of UK Weather available
  2. Whether a different URL should be used for obtaining the data from the Internet
  3. Whether, if a new version of UK Weather is found, the user should be notified and given the option to download the new version.
The "Notify user if new version is found" check box allows you to decide whether or not you want to be notified in the event that a new version is found. If a new version is found and there is a tick in this box, the following popup will appear:



Of course, the numbers will be different. Checking the "Don't ask me again" box, followed by either of the two buttons, has the same effect as unchecking the "Notify user if new version is found" check box. From here, there are two options.
  1. Download = Go to the web page that has the new version. Note that as of version 0.6, the software is not automatically downloaded or installed (though this may be a feature of a later version).
  2. Ignore = Ingore the warning and continue as normal.
