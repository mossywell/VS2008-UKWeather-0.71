Installation Details

This section explains what is installed and where. It is useful if you wish to uninstall the software manually, for example.

Desktop Changes

The setup program only adds an icon to the desktop. The start menu is left untouched. So, if you want to add the item to the start menu, you will have to add it manually.

Installed Files

The default installation directory is:
  • C:\Program Files\Mossywell\UK Weather
As of version 0.60, the files installed into this directory are:

File Details
README.TXT Last minute information about the application not present in the help pages.
UKWeather.exe The application itself. Note that it is a standalone executable and doesn't come with any DLLs.
UKWeather.chm This help file!

Registry Settings

As of version 0.50, the Class IDs used by this application are:
  • {05120F5A-9D53-4C7B-80D2-B307EFAAAB93} - used by the installer
Application settings are stored in the following location:
  • HKEY_CURRENT_USER\Software\Mossywell\UK Weather
The individual values are obvious because they're taken directly from the interface, so they won't be explained further here. Note that XML files are not used to store settings (which is Microsoft's preferred method in .Net applications) because XML settings are not dynamic and only take effect when the application is restarted.

If the "Run at system startup" option is chosen, it stores the setting in the following location:
  • HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run
in a value (not surprisingly) called "UKWeather". Note that I'm using the current user rather than HKEY_LOCAL MACHINE hive.

Finally, the Windows Installer saves a plethora of settings. They're not listed here, but a trawl for the word "mossywell" in the registry will dig them out.
